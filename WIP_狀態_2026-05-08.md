## 🧠 已踩雷紀錄(不要再犯)

### 雷 1:Vue 3 method vs computed
**症狀**:template 用 `xxx.kind` 直接讀,但 xxx 是 method → 拿到 function 屬性 (永遠 undefined)
**規則**:**屬性訪問用 computed,事件處理才用 method**
**已修**:licenseStatus / isReadOnly / licenseDebugInfo 都搬到 computed

### 雷 2:Firestore onSnapshot 不能用 length > 0 過濾
**症狀**:雲端清空時,本地不會更新 (跟薪資系統同樣的雷)
**規則**:onSnapshot 直接覆蓋本地,雲端是真相來源
**例外**:adminUsers 可保留 length > 0 (避免空雲清掉本地預設帳號)

### 雷 3:GitHub Pages 子目錄絕對路徑會 404
**症狀**:`<img src="/assets/imgs/xxx.jpg">` 在 `chanel5168-cell.github.io/yangji-shop/` 下載不到圖
**規則**:GitHub Pages 子目錄部署 → 用相對路徑 `assets/imgs/xxx.jpg`(去掉開頭 `/`)
**已修**:加 `migrateImagePaths()` 一次性遷移函式

### 雷 4:Bash 環境的 SSL 憑證問題
**症狀**:Claude 的 Bash 工具 curl 到 GitHub api 失敗 (CRYPT_E_NO_REVOCATION_CHECK)
**規則**:git push 必須在 PowerShell 跑(Windows 系統認證才會運作)
**位置**:`PS C:\Users\User\Desktop\我的商城> git push`

### 雷 5:Firebase 首次連線會用空雲覆蓋本地
**症狀**:設定好 firebaseConfig 重整 → 本地資料被空雲清掉
**規則**:`bootstrapFirebaseFromLocal()` 在訂閱前先把本地推上去
**flag**:`shop_firebase_bootstrapped_v1` 確保只跑一次

### 雷 6:LINE 設定欄位不能貼整段 HTML
**症狀**:把 LINE Official Account 給的 `<a href="..."><img></a>` 整段貼到「LINE 官方帳號 ID」欄
**規則**:
- LINE 官方帳號 ID:只填 `@305xxlkn` (8-9 碼)
- 加好友連結:只填 `https://lin.ee/wx8M7bNt`
